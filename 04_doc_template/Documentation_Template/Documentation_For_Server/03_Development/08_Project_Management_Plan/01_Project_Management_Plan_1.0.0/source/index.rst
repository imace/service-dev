Welcome to Service Dev's documentation!
=====================================================================

..
  该计划适用于项目经理制定软件开发的项目管理计划。项目管理计划制定完后可以发布给所有项目成员。
  该计划根据PMBOK指南，结合实际工作需要进行了订制。在实际工作中，也可以根据需要添加或者删除计划中的相关内容。

  该计划中主要包括以下内容：
  01_项目背景：介绍项目开展的背景
  02_项目目标（范围）：定义项目的目标（范围）
  03_项目计划：制定项目的里程碑计划和详细计划
  04_项目验收标准：定义项目完成的标准
  05_项目风险：定义项目风险，并记录风险状态
  06_项目沟通：定义项目沟通计划和策略
  07_项目人员：列举项目干系人
