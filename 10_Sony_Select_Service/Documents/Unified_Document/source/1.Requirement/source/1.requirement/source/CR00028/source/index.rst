CR00028 消息推送实现部署外链功能
====================================

**需求明细:**


#. 功能位置：消息管理->消息管理->‘新增’
#. 加入部署外链品牌产品功能（sony品牌产品）
#. 新增推送内容，下拉选择框，内容包括：文本；外链
#. 选择文本时，与原功能一致；选择外链时，弹出新必填项及选填项
#. 必填项：外链地址、消息标题、消息内容、生效时间、失效时间
#. 选填项：Icon图片（做上传按钮）、消息描述
#. 详情请见示意图辅助说明需求5
